### 1. Install the following plugin
vagrant plugin install vagrant-vbguest


### 2. Commnands

* How to start a box:
```
vagrant up box-name
```
* How to stop a box:   
```
vagrant halt box-name
```
* How to remove a box:
```
vagrant destroy box-name
```
* How to connect to a box:
```
vagrant ssh box-name
```
